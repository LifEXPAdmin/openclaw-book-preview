# OpenClaw Book Companion Site

This site is the living companion to the book manuscript.

## Purpose
- Host copy/paste templates
- Host prompt packs
- Provide interactive setup guides
- Publish updates and new features not in the print/PDF edition

## Run locally
From this folder:

```bash
python3 -m http.server 8080
```

Then open: <http://localhost:8080>

## Content policy
Any time the book is updated, update this site.
Any time OpenClaw changes and the book lags, update this site first and add to “What’s New Since the Book”.
